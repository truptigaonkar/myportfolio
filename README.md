## Portfolio Sample - [Live Demo]( https://truptigaonkar.github.io/myportfolio/)
#### Technologies
* Node
* React hooks
* Typescript
* Glitz (https://github.com/frenic/glitz#getting-started)

#### Installation
```
  $ git clone https://github.com/truptigaonkar/myportfolio.git
  $ cd myportfolio.git
  $ npm install
  $ npm start
```
